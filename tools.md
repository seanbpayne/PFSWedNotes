## Token Mapping - Same
Blind -> blind
Deafened -> Deafened
Dying -> dying
Frightened -> frightened
Grappled -> grabbed
Invisible -> invisible
Paralized -> paralized
Petrified -> petrified
Prone -> prone
Restrained -> restrained
Stunned -> stunned
Unconscious -> unconscious

## Token Mapping -> different
Bleeding -> none
Burning -> persistent damage
Exhausted -> fatigued
Incapacitated -> Immobalized
Poisoned -> sickened
